# Генератор текста в стиле Льва Толстого

## Идея проекта

Обучить нейросеть генерировать текст в стиле Льва Толстого. Модель получает начало фразы и продолжает её, подражая манере письма великого русского писателя.

---

## Сбор данных

Для обучения взяты самые популярные произведения Льва Толстого на русском языке в формате txt:

- Война и мир
- Анна Каренина
- Детство

---

## Предобработка данных

1. Регулярными выражениями удалены:
   - маркеры диалогов (—, -, *)
   - иностранные слова и цифры
   - специальные символы

2. Оставлены только русские буквы, точки и запятые

3. Выполнена посимвольная токенизация — каждый символ превращён в число

Размер словаря: 118 уникальных символов

---

## Модель

### Архитектура

Использована архитектура **Decoder-only Transformer** (как в GPT). Это стандартный выбор для генерации текста: модель предсказывает следующий символ на основе предыдущих.

### Компоненты

**1. Embedding слой**
- Превращает индексы символов в вектора размером 128
- Помогает модели находить связи между символами

**2. Позиционное кодирование**
- Добавляет информацию о порядке символов
- Использует синусоидальное кодирование из оригинального Transformer

**3. Три блока декодера**

Каждый блок содержит:

- **Multi-Head Attention (4 головы)** — модель смотрит на предыдущие символы и определяет, какие из них важны для предсказания следующего. Causal mask запрещает заглядывать в будущее.

- **Feed-Forward Network** — два линейных слоя (128 → 512 → 128) с активацией GeLU. Добавляет нелинейность и преобразует информацию.

- **Residual connections** — пропускают вход напрямую к выходу, ускоряют обучение

- **Layer Normalization** — стабилизирует обучение

**4. Выходной слой**
- Линейный слой, который превращает скрытое представление обратно в вероятности для каждого из 118 символов

### Параметры модели

| Параметр | Значение |
|----------|---------|
| Размер словаря | 118 символов |
| Размер эмбеддингов | 128 |
| Количество голов | 4 |
| Количество слоёв | 3 |
| Размер скрытого слоя | 512 |
| Dropout | 0.1 |


### Обучение

| Параметр | Значение |
|----------|---------|
| Функция потерь | CrossEntropyLoss |
| Оптимизатор | AdamW |
| Learning rate | 0.0003 |
| Длина контекста | 128 символов |
| Batch size | 64 |
| Эпох | 20 |
